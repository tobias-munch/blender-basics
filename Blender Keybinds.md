Hvad man skal trykke på for at bruge Blenders forskellige funktioner
## Object Mode

| Keybind | Funktion                       |
| ------- | ------------------------------ |
| G       | [[Flytning (Move)]]                   |
| R       | [[Rotation (Rotate)]]                   |
| S       | [[Skaléring (Scale)]]                  |
| X       | Slet                           |
| A       | Vælg alle                      |
| SHIFT+A | [[Tilføj objekt (Add Object)]] |
| SHIFT+D | [[Lav kopi af objekt]]         |
